## How to run this project
Clone this repo

```bash
git clone https://github.com/LucsGomes/HelpDogApp
```
Go to project directory
```bash
cd HelpDogApp
```
Install the dependencies
```bash
npm install | npm i
```
Start the aplication
```bash
npm run start
```
See in your device using Expo Go
